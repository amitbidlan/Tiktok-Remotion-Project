# MojiIQ Remotion Video Renderer

Automated TikTok lesson video renderer for [MojiIQ by Zistica](https://mojiiq.zistica.com).

## How It Works

1. `content/lesson.json` contains all lesson content (filled by Claude)
2. GitHub Actions runs automatically on `workflow_dispatch`
3. Downloads Pexels background videos
4. Generates ElevenLabs voiceover audio
5. Renders 1080×1920 MP4 with Remotion
6. Uploads MP4 as downloadable artifact

## Setup (one time)

### 1. Add GitHub Secrets
Go to: **Repo Settings → Secrets and variables → Actions**

| Secret | Value |
|--------|-------|
| `PEXELS_API_KEY` | Your Pexels API key |
| `ELEVENLABS_API_KEY` | Your ElevenLabs API key |

### 2. Make repo public (for unlimited free Actions minutes)
Go to: **Repo Settings → Danger Zone → Change visibility → Public**

## Rendering a Video

1. Push updated `content/lesson.json` to `main`
2. Go to **Actions tab → Render MojiIQ Lesson Video → Run workflow**
3. Enter a lesson title (used for the artifact name)
4. Wait ~10 minutes
5. Download MP4 from the artifact section
6. Delete artifact after download to free storage

## Local Development

```bash
npm install
npm run fetch-videos   # downloads Pexels videos
npm run generate-audio # generates ElevenLabs audio
npm start              # opens Remotion Studio at localhost:3000
npm run render         # renders MP4 to out/mojiiq-lesson.mp4
```

## Project Structure

```
├── .github/workflows/render.yml   ← GitHub Actions pipeline
├── content/
│   ├── lesson.json                ← lesson content (updated per video)
│   └── audio-manifest.json        ← auto-generated by generate-audio.js
├── public/
│   ├── videos/                    ← auto-downloaded by fetch-videos.js
│   └── audio/                     ← auto-generated by generate-audio.js
├── scripts/
│   ├── fetch-videos.js            ← Pexels video downloader
│   └── generate-audio.js          ← ElevenLabs audio generator
├── src/
│   ├── index.ts                   ← Remotion entry point
│   ├── Root.tsx                   ← composition registration
│   ├── MojiiqLesson.tsx           ← main video composition
│   └── slides/                    ← individual slide components
└── package.json
```
